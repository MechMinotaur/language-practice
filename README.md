# language-practice
A repo I use to demonstrate language competency.

## Demonstration Goals
These are the goals I set to accomplish in each language.

1. Import a function from a separate file.
1. Import functionality from an open source package or library I did not create.
1. Create the equivalent to a Python lambda function.
1. Practice Inversion of Control through Dependency Injection.
1. Demonstrate knowledge of Model View Controller architecture.
1. Create, read from, and write to an sqlite database.
1. Write a unit test.
1. Communicate with a socket.
1. Prove knowledge of source control (this repo).
